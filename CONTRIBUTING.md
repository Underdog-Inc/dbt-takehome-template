# Contributing

This repository is a **take-home assessment template** used by Underdog for analytics engineering candidate evaluations. It is published publicly for licensing reasons; it is not an open-source project soliciting contributions.

## For candidates

- Do **not** open a pull request against this repository.
- Follow the instructions in [TASK.md](TASK.md): fork or template this repo into your own private repository and submit your work there.
- If you have a question about the assessment itself, contact your Underdog point of contact directly — please do not open issues here for assessment questions.

## Bug reports and typos

If you spot a clear bug or typo in this template (broken link, incorrect setup instruction, etc.), feel free to open an Issue. We do not accept external pull requests; we'll triage and address valid reports ourselves.

## Maintainers

Maintained by the Underdog analytics engineering team.
